# FUTURE_CS_03

API Security Risk Analysis - Task 3

**Overview**

This project analyzes security risks in a public API using Postman.

**Tools Used**

Postman
JSONPlaceholder API
Browser DevTools

**Key Findings**

No authentication vulnerability
Excessive data exposure
No rate limiting
Weak input validation

**Conclusion**

The API contains multiple security risks that could lead to data breaches and system abuse if implemented in a real-world application.
